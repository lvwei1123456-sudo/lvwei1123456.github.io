# 临床试验平台演示系统

## 文件说明

### 主入口页面
- **index.html** - 演示系统主入口页面，提供项目管理系统入口

### 项目管理系统
- **project-demo.html** - 项目列表页面，演示新建项目流程
- **protocol-view.html** - 试验方案查看页面，展示入组标准和排除标准
- **screening-rules.html** - 筛选规则配置页面，支持上传文档识别和手动配置

## 使用说明

### 项目管理系统演示流程

1. **打开项目列表** (`project-demo.html`)
   - 查看现有项目列表
   - 点击"新建项目"按钮

2. **新建项目**
   - 填写项目信息（已自动填充演示数据）
   - 点击"确定"创建项目

3. **查看试验方案**
   - 在项目列表中点击"试验方案"列的"查看"链接
   - 进入试验方案页面，查看入组标准和排除标准

4. **配置筛选规则**
   - 在试验方案页面点击"筛选规则"按钮
   - 点击"上传研究文档或粘贴筛选条件"按钮
   - 选择文件后点击"开始识别"
   - 识别完成后，点击"确认"按钮返回试验方案页面
   - 规则数据会自动填充到入组标准和排除标准面板

5. **下载配置文件**
   - 在试验方案页面点击"下载配置文件"按钮
   - 自动下载加密的配置文件

## 功能特点

- ✅ 完全按照设计图实现，样式和结构保持一致
- ✅ 所有按钮和链接均可点击
- ✅ 演示数据自动填充，无需手动输入
- ✅ 页面间跳转流畅，数据自动传递
- ✅ 支持文件上传和识别功能演示
- ✅ 支持配置文件下载（加密格式）

## 技术说明

- 纯HTML + CSS + JavaScript实现
- 无需服务器，可直接在浏览器中打开使用
- 所有数据存储在URL参数中，页面刷新会保留数据
- 使用Base64编码实现配置文件加密

## 浏览器兼容性

- Chrome（推荐）
- Firefox
- Safari
- Edge

## 注意事项

- 所有文件需要在同一目录下
- 建议使用现代浏览器打开
- 文件上传功能为演示，不会真正上传文件
- 识别功能为模拟，会自动填充演示数据
